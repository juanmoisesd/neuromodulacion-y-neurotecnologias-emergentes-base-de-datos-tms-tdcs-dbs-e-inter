# Reproducibility Guide

## Dataset: Repositorio de bases de datos, registros y repositorios sobre neuromodulación y neurotecnologías emergentes: estimulación magnética transcraneal (TMS), estimulación transcraneal por corriente directa (tDCS), estimulación cerebral profunda (DBS), inte

---

## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/neuromodulacion-y-neurotecnologias-emergentes-base-de-datos-tms-tdcs-dbs-e-inter.git
cd neuromodulacion-y-neurotecnologias-emergentes-base-de-datos-tms-tdcs-dbs-e-inter
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
- See `data_dictionary.csv` or README for variable descriptions
