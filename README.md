## Overview
Repositorio de bases de datos, registros y repositorios sobre neuromodulación y neurotecnologías emergentes: estimulación magnética transcraneal (TMS), estimulación transcraneal por corriente directa (tDCS), estimulación cerebral profunda (DBS), interfaces cerebro-computadora (BCI), y resultados cognitivos, clínicos y neurofisiológicos asociados.
